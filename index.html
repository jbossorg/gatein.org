JBoss Portlet Bridge

Build new JSF applications or embed existing JSF applications in GateIn projects with the JBoss Portlet Bridge.

Final version of PortletBridge 2 supports JSF 1.2 and RichFaces 3 to build portlets. Version 3.0 is made to support JSF2 while 3.1 versions will also support RichFaces 4.
