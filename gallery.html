GateIn Portal - Gallery

Homepage

The home page of GateIn Portal

Signing-in

When signing in, the portal will deliver content adapted to the user connecting to the website.

The application Registry

The application registry helps classifying applications made available to build a page

Navigation Node management

Content managers can manipulate navigation nodes through this interface and reorganize the menus

Page Wizard (1/3)

To build a new page, a wizard is available, first screen is about the description of the page and its availability.

Page Wizard (2/3)

The second step of the wizard lets you start from an existing template, but the 32rd step will let you customize it.

Page Wizard (3/3)

The third step lets you rearrange how to split a page and which applications to put on it.

Layouts

If parts of the page should be on all the pages (such as a footer and header) it can be part of the layout.

User & Group Management

User with correct privileges will be able to edit users and groups unless it is preferred to use an LDAP provisioning tool.

Skins

Websites can have different skins, those are deployable components and makes it easy to change the graphical aspect of a website.

Internationalization

GateIn portal makes it easy to support multiple languages including Right to Left languages.
